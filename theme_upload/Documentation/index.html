HTML Structure

Jason follows a simple and easy to customize coding structure. Here is the sample for your reference:
The template is based on Bootstrap Framework

                
<!DOCTYPE html>
<html lang="en" data-theme="light">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
    <title>Find Trip - Travel tour booking adventure agency</title>

    <link href="/" rel="canonical">
    <link href="assets/img/favicon.png" rel="shortcut icon" type="image/vnd.microsoft.icon">

    <!-- start style -->
    <link href="assets/build/css/main.min.css" media="screen" rel="stylesheet" type="text/css">
    <!-- end style -->
</head>
<body class="home">

    <!-- start loading-->
    <div class="loading"></div>
    <!-- end loading-->

    <!-- start header-->
    <section class="header d-flex align-content-center flex-row"></section>
    <div class="header--fixed"></div>
    <!-- end header-->

    <!-- start hero-->
    <section class="hero hero--form" data-viewport="opacity"></section>
    <!-- end hero-->

    <!-- start discount -->
    <section class="discount"></section>
    <!-- start discount -->

    <!-- start support-->
    <section class="support"></section>
    <!-- end support-->

    <!-- top destinations -->
    <section class="top-destinations"></section>
    <!-- end top destinations -->

    <!-- start resort surprises -->
    <section class="resort-surprises"></section>
    <!-- end resort surprises -->

    <!-- start explore destinations -->
    <section class="explore-destinations"></section>
    <!-- end explore destinations -->

    <!-- start popular search -->
    <section class="popular-search"></section>
    <!-- end popular search -->

    <!-- start my blog -->
    <section class="blog"></section>
    <!-- end my blog -->

    <!-- start better work -->
    <section class="better-work" data-viewport="opacity"></section>
    <!-- end better work -->

    <!-- start footer -->
    <section class="footer mt-0 mt-sm-5 text-center text-sm-start"></section>
    <!-- end footer -->

    <!-- start script -->
    <script src="assets/build/js/main.bundle.js"></script>
    <!-- end script -->

</body>
</html>

If you need more information, please visit bootstrap site: https://getbootstrap.com

Development

Prerequisites

Note you have to install node and npm
When we develop this project it is used for version: Node 14.17.6 & NPM 6.14.15

1. In the project directory HTML, you can run:

npm install

2. Launches the test runner in the interactive watch mode.

npm run watch

3. Builds the app for production to the build folder.

npm run build
