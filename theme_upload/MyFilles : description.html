Overview

Tripf - Is a template designed with a modern style. It’s a modern, creative, clean, professional, attractive travel agency template.

Template is developed on bootstrap 5.1.3, using webpack, scss optimized compress library.

Responsive, retina display, lazy loading are also used in themes.

Template is very customizable, with notes on each component, for style and javascript
